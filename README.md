# Rocketfy

This project is a [Pipefy](https://www.pipefy.com/) kanban function clone developed from [Rocketseat's class](https://www.youtube.com/watch?v=awRtgpRsdTQ&t=801s)

<img src="rocketfy.png">


# Installation

```
npm install
npm start
```` 

# Built with

* React.js
* React Hooks
* React Drag n Drop
* Immer
* Styled Components
